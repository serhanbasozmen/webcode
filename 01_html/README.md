# HTML Web Temelleri | HTML Web Fundamentals 🌐

## 🇹🇷 Türkçe

Web geliştirme serüvenimin temeli olan HTML etiketleri, sayfa iskeleti oluşturma ve anlamsal (semantic) web pratiklerimi içerir.

### İçerik

- **Temel Etiketler:** Metin biçimlendirme, başlıklar (h1-h6), paragraflar, linkler ve görseller.
- **Semantik Yapı:** Arama motoru dostu sayfa tasarımı (header, nav, section, article, footer).
- **Veri ve Etkileşim:** Kullanıcıdan veri almak için form elemanları ve veri gösterimi için tablolar.

## 🇬🇧 English

Includes my fundamental HTML tags, page skeleton creation, and semantic web practices, which are the foundation of my web development journey.

### Content

- **Basic Tags:** Text formatting, headings (h1-h6), paragraphs, links, and images.
- **Semantic Structure:** Search engine friendly page design (header, nav, section, article, footer).

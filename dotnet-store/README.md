## 🇹🇷 Türkçe (Turkish)

**Dotnet Store**, modern web teknolojileri kullanılarak geliştirilmiş kapsamlı bir e-ticaret projesidir. Temel amacı, kullanıcıların ürünleri listeleyebileceği, inceleyebileceği ve güvenli bir şekilde alışveriş yapabileceği bir altyapı sunmaktır.

### Kullanılan Teknolojiler

- **Backend:** ASP.NET Core, C#
- **Veritabanı:** Entity Framework Core, SQL Server
- **Frontend:** HTML, CSS, JavaScript (veya entegre edilen modern frameworkler)

---

## 🇬🇧 English

**Dotnet Store** is a comprehensive e-commerce project developed using modern web technologies. Its main purpose is to provide an infrastructure where users can list, review, and securely purchase products.

### Technologies Used

- **Backend:** ASP.NET Core, C#
- **Database:** Entity Framework Core, SQL Server
- **Frontend:** HTML, CSS, JavaScript (or integrated modern frameworks)

---

### 🚀 Kurulum / Installation

**[TR]** Projeyi kendi bilgisayarınızda çalıştırmak için:

1. Repoyu klonlayın: `git clone https://github.com/serhanbasozmen/dotnet-store.git`
2. `appsettings.json` içindeki veritabanı bağlantı cümlenizi (connection string) kendi SQL Server'ınıza göre güncelleyin.
3. Package Manager Console (veya terminal) üzerinden `Update-Database` komutunu çalıştırarak tabloları oluşturun.
4. Projeyi ayağa kaldırın.

**[EN]** To run the project locally:

1. Clone the repo: `git clone https://github.com/serhanbasozmen/dotnet-store.git`
2. Update the connection string in the `appsettings.json` file according to your local SQL Server.
3. Run the `Update-Database` command via Package Manager Console (or terminal) to create the tables.
4. Build and run the project.
